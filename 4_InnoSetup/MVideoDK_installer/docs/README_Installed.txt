MVideoDK — Documentación (Instalado)

Gracias por instalar MVideoDK.

Este proyecto es personal y educativo (no comercial) y se comparte para aprendizaje
y mejoras comunitarias.

==================================================
INICIAR
==================================================

Ejecuta:
  MVideoDK.exe

El launcher muestra un splash e inicia el Core.

==================================================
RUTAS IMPORTANTES
==================================================

Programa:
  C:\Program Files\MVideoDK\

Documentación instalada:
  C:\Program Files\MVideoDK\docs\

Recursos persistentes:
  C:\ProgramData\MVideoDK\

Recursos opcionales para el usuario:
  %USERPROFILE%\Downloads\MVideoDK_resources\
    - Extension\
    - Apk\

==================================================
EXTENSIÓN (OPCIONAL)
==================================================

La extensión es opcional. MVideoDK funciona sin ella.

Para instalarla (Chrome / Edge):
  1) Abre chrome://extensions (o edge://extensions)
  2) Activa "Modo desarrollador"
  3) "Cargar descomprimida"
  4) Selecciona:
     %USERPROFILE%\Downloads\MVideoDK_resources\Extension\

==================================================
SI EL PROGRAMA NO INICIA
==================================================

Verifica que en:
  C:\Program Files\MVideoDK\

estén:
  - MVideoDK_core.exe
  - la carpeta _internal

Y verifica que existan herramientas en:
  C:\ProgramData\MVideoDK\bin\
